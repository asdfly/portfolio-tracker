## 二、改进方向与功能设计

### 2.1 上海金基准价对比模块（新增）

**目标**：将上海金基准价与SGE现货价格进行对比分析，揭示价差规律。

**功能设计**：
- 双轴折线图：上海金基准价 vs SGE Au99.99收盘价
- 价差柱状图（基准价 - 现货价），标注均值线和±1σ区间
- 统计指标卡片：平均价差、标准差、当前价差分位数

**数据源**：`spot_golden_benchmark_sge` + `spot_hist_sge`

**实现复杂度**：低 — 数据源已验证可用，仅需数据对齐和图表绑定

---

### 2.2 定价因子相关性分析模块（新增）

**目标**：可视化黄金核心定价因子与金价的相关性，辅助宏观判断。

**核心定价逻辑**：
- **实际利率**（名义利率 - 通胀预期）← 负相关，黄金核心定价锚
- **美元指数** ← 负相关（约-0.72），全球黄金以美元计价
- **中美利差** ← 影响人民币金价和跨境资金流
- **CPI/通胀** ← 影响实际利率，间接影响金价

**功能设计**：
- **相关系数热力图**：金价 vs 美元指数/实际利率/中美利差/CPI，计算Pearson相关系数矩阵
- **双轴趋势对比图**：金价叠加定价因子（可切换），直观展示负相关/正相关走势
- **散点图矩阵**（可选）：各因子与金价的散点分布+拟合线

**数据源**：
- 金价：`spot_golden_benchmark_sge`（上海金基准价，日频，适合与宏观因子对齐）
- 美元指数：`akshare.index_us_dxy` 或 `macro_usa_dxy`（需验证可用性）
- 中美利差：`bond_zh_us_rate`
- CPI：`macro_china_cpi`

**实现复杂度**：中 — 需要多数据源日期对齐（频率不同：日频vs月频）、相关系数计算、热力图渲染

---

### 2.3 黄金季节性规律分析模块（新增）

**目标**：基于历史数据统计黄金按月涨跌概率和平均收益率，揭示季节性规律。

**统计规律（近22年）**：
- 2月、8月、9月： historically偏强月份
- 3月、5月： historically偏弱月份（5月平均收益率为负）
- 10月-12月：年底通常有季节性买盘

**功能设计**：
- **季节性热力图**（日历视图）：横轴月份，纵轴年份，颜色深浅表示月涨跌幅
- **月度统计柱状图**：各月的平均收益率 + 上涨概率（%），双色柱
- **当前月份位置标注**：在热力图中高亮当前月份，提示历史表现

**数据源**：`spot_hist_sge`（历史K线，计算月度收益率）

**实现复杂度**：低 — 纯统计计算 + Plotly热力图，数据源已可用

---

### 2.4 黄金技术信号面板（新增）

**目标**：集成MACD+RSI+Bollinger三指标组合，提供黄金专属技术分析。

**最佳实践组合**（调研结论）：
- **MACD**（12,26,9）：趋势方向确认
- **RSI**（14）：超买超卖判断（>70超买，<30超卖）
- **Bollinger Bands**（20,2）：波动率和价格通道

**功能设计**：
- 技术指标计算函数（`ta`库或手写）
- **三合一技术图**：
  - 主图：金价K线 + 布林带上/中/下轨
  - 副图1：MACD线 + 信号线 + 柱状图
  - 副图2：RSI线 + 超买/超卖参考线
- **综合信号指标**：基于三指标综合判定（多头/空头/中性），显示在标题栏
- 时间周期可切换：近3月/6月/1年/3年

**数据源**：`spot_hist_sge`（已有）

**实现复杂度**：中 — 需要技术指标计算逻辑，但数据源现成

---

### 2.5 外汇储备与黄金储备占比分析（增强）

**目标**：在现有中国黄金储备图表基础上，增加外汇储备占比和增持趋势分析。

**功能设计**：
- **黄金储备占外汇储备比例**折线图（国际比较视角，中国目前约4.9%，全球平均约15%）
- **增持速度分析**：季度/年度增持量柱状图，标注关键政策节点
- 与SPDR持仓变化的对比图（东西方黄金流动）

**数据源**：`macro_china_fx_gold`（已有，新增使用外汇储备列）+ `macro_cons_gold`（已有）

**实现复杂度**：低 — 数据源已有，仅需新增计算和图表

---

### 2.6 SGE多品种实时行情面板（增强）

**目标**：从单一品种扩展到SGE主要品种实时行情一览。

**功能设计**：
- 可交互表格/卡片：展示SGE 17个品种的实时行情（品种名、最新价、涨跌幅、成交量等）
- 按"投资属性"分组：标准合约(Au99.99/Au100g) / 迷你合约(mAu99.99) / 国际板(iAu99.99)等
- 点击品种可切换该品种的K线走势图

**数据源**：`spot_symbol_table_sge`（品种表）+ `spot_quotations_sge`（实时行情）

**实现复杂度**：低-中 — 数据源已验证，需要UI交互逻辑

## 四、技术架构建议

### 4.1 模块拆分

建议将Tab11拆分为子组件，保持代码可维护性：

```
tabs/
├── tab11_gold.py                 # Tab入口，子Tab布局
└── gold_components/
    ├── price_comparison.py       # 上海金基准价对比
    ├── seasonality.py            # 季节性规律分析
    ├── technical_signals.py      # 技术信号面板（MACD+RSI+Bollinger）
    ├── correlation.py            # 定价因子相关性分析
    ├── reserve_analysis.py       # 储备分析（黄金+外汇）
    ├── realtime_quotes.py        # SGE多品种实时行情
    └── gold_utils.py             # 公共工具函数（数据获取、指标计算等）
```

### 4.2 数据缓存策略

- 黄金相关数据日频更新即可，建议使用`st.cache_data(ttl=3600)`（1小时缓存）
- 历史数据（季节性分析需要多年数据）首次加载后可长缓存
- 实时行情数据短缓存（5-15分钟）

### 4.3 数据源验证待办

| 数据源 | 需验证接口 | 验证内容 |
|--------|-----------|---------|
| 美元指数 | `akshare` 相关接口 | 是否可用、数据频率、历史深度 |
| 实际利率 | 需组合计算（国债收益率-CPI） | 中美国债收益率接口可用性 |
