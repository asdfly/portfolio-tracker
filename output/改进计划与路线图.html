一、项目概况

投资组合跟踪分析系统（Portfolio Tracker）是一套基于 Python + Streamlit 的 ETF 投资组合实时监控与量化分析平台。系统从通达信导出持仓数据，通过新浪财经/K线接口获取行情，存储至 SQLite 数据库，并在 Streamlit Dashboard 中以 Dark Theme 风格展示。

Dashboard 初始基线版本（commit e5829f0）提供基础的四标签页布局和核心指标展示。通过参考 QuantStats、pyfolio、Interactive Brokers PortfolioAnalyst 等业界标杆工具，制定了分三级（P0/P1/P2）的渐进式改进计划。

全部改进已完成交付。P0（7项）→ P1（10项）→ P2（4项），共 21 项改进，Dashboard 从基线版本扩展至 5 个 Tab / 2568 行 / 30 个函数。

三、优先级分级原则

优先级 定义 投入产出比
P0 基于现有数据即可计算的关键指标与可视化，不依赖新增数据源或外部接口 高 — 代码改动小，用户感知大
P1 需要新增数据函数和 UI 组件的功能扩展，涉及交互逻辑优化 中高 — 代码改动中，显著提升分析深度
P2 高级分析能力和数据输出，引入新的计算逻辑或外部工具 中 — 代码改动大，专业性强
